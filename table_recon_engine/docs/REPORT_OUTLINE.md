# 基于纯视觉特征与规则推断的无边框表格识别及 LaTeX 结构重建

## 1. 选题背景与意义

说明文档智能化、无边框表格识别难点，以及纯视觉路线相对 PDF 文本解析路线的适用场景。

## 2. 核心算法设计与实现机制

### 2.1 基于 YOLO 的表格结构检测

说明使用开源标注数据转 YOLO 格式，训练六类表格结构检测器。

### 2.2 无边框表格拓扑重建算法

重点写 `CellCluster`、`GridBuilder`、`LaTeXGenerator` 三个手写模块。

- `CellCluster`：基于中心点排序、水平/垂直投影重合度、动态阈值完成行列聚类。
- `GridBuilder`：根据聚类中心生成虚拟网格，填充缺失位置，推断 colspan/rowspan。
- `LaTeXGenerator`：遍历二维网格，生成 `tabular`、`&`、`\\`、`\multicolumn`、`\multirow`。

### 2.3 非几何图像退化鲁棒性测试

说明高斯噪声、高斯模糊、亮度/对比度变化、JPEG 压缩用于模拟扫描或拍照质量下降。其中高斯模糊由手写高斯卷积核实现，不引入旋转、透视等几何变换。
